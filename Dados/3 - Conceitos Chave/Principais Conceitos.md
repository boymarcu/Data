- Análise de dados é o processo de inspecionar, limpar, transformar e modelar dados para descobrir informações úteis e apoiar a tomada de decisões.
  
- Os dados podem ser classificados em vários tipos, como nominal, ordinal, intervalo e razão, cada um com uma função e técnica de análise específicas.
  
- Qualidade e o gerenciamento de dados são conceitos-chave para garantir conjuntos de dados limpos e confiáveis
