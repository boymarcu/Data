##### PROCV // PROCH

-  Função versáteis de pesquisa e referência que pode encontrar dados específicos em uma vasta variedade, proporcionando facilidade e conveniência em tarefas de recuperação de dados.
  
- ###### PROCV
	- Usada para encontrar dados em uma tabela classificada **verticalmente**.

- ###### PROCH
	- Usada para encontrar dados em uma tabela classificada **horizontalmente**.
