###### - Se (if)
	Cria instruções condicionais;
	Limpa e valida dados;
	Realiza cálculos com base em condições específicas;
	Cria métricas personalizadas;
	Aplica formatação condicional;
	Atomatiza tarefas;
	Gera relatórios dinâmicos.

- Analistas de dados usam o IF para:
	- Categorizar dados;
	- Lidar com valores ausentes;
	- Calcular bônus ou métricas personalizadas;
	- Destacar tendências;
	- Aprimorar visualizações.
